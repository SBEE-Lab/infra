# 접근 권한 요청

## 필요한 정보

관리자에게 다음 정보를 전달합니다:

| 항목 | 예시 | 필수 |
|------|------|------|
| 이름/사용자명 | `jdoe` | O |
| SSH 공개키 | `ssh-ed25519 AAAA...` | O |
| 역할 | 연구원/학생/관리자 | O |
| 만료일 | `2026-08-31` | 학생/연구원 |

## 사용자 유형

| 유형 | 서버 접근 | 서비스 접근 | Headscale ACL |
|------|----------|-----------|---------------|
| **관리자** | 모든 호스트 | 전체 | `sjanglab-admins` |
| **연구원** | 지정 호스트 | AI + 앱 | `sjanglab-researchers` |
| **학생** | 지정 호스트 | 앱만 | `sjanglab-students` |

## 계정 생성 후

관리자가 계정을 생성하고 배포하면 다음을 받게 됩니다:

1. **Authentik 계정** — SSO 로그인용 (이메일로 초대)
1. **SSH 접근** — 제출한 공개키로 자동 설정
1. **Headscale 그룹** — 역할에 따라 자동 배정

다음 단계: [VPN 설정](vpn-setup.md)
