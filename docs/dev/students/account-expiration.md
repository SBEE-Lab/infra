# 계정 만료

학생 및 연구원 계정에는 `expires` 필드가 설정됩니다 (예: `2026-08-31`).

## 만료 전

- 만료일 전에 [접근 권한 요청 채널](../../guide/requesting-access.md#access-request-channel)로 연장을 요청하세요
- 중요한 데이터는 미리 로컬로 백업하세요

## 만료 시

- SSH 로그인이 차단됩니다
- Headscale/Authentik 접근도 그룹에서 제거되면 차단됩니다
- `/project/<username>/` 데이터는 즉시 삭제되지 않지만, 일정 기간 후 정리됩니다

## 연장 요청

접근 권한 요청 채널로 연장을 요청하면 관리자가 만료일을 갱신한 뒤 계정을 복구합니다.
