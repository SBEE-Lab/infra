CI/CD¶

Nixbot¶

https://buildbot.sjanglab.org вҖ” GitHub м—°лҸҷ Nix CI/CDмһ…лӢҲлӢӨ. лҸ„л©”мқёкіј check contextлҠ” кё°мЎҙ GitHub branch protectionмқ„ мң м§Җн•ҳкё° мң„н•ҙ buildbot мқҙлҰ„мқ„ кі„мҶҚ мӮ¬мҡ©н•©лӢҲлӢӨ.

кө¬м„ұ¶

flowchart LR
  gh["GitHub App мӣ№нӣ…"] --> edge["public nginx + TLS<br/>eta :443"]
  edge --> proxy["Nixbot nginx upstream<br/>psi wg-admin :443"]
  proxy --> svc["nixbot<br/>psi"]
  svc --> db["PostgreSQL<br/>psi"]
  svc -- "nix-eval-jobs вҶ’ nix build" --> nix["local nix daemon<br/>psi"]
  nix --> result["л№Ңл“ң кІ°кіј"]
  result --> gh
  • Service host: psi (nixbot.service)
  • Public reverse proxy/TLS: eta (нҸ¬нҠё 443), wg-adminмңјлЎң psi nginxм—җ н”„лЎқмӢң
  • DB: PostgreSQL (psi, local peer auth)
  • Build execution: psiмқҳ local nix daemon
  • Check context prefix: buildbot (buildbot/nix-eval, buildbot/nix-build ...)

л№Ңл“ң нҠёлҰ¬кұ°¶

  • GitHub App-level мӣ№нӣ…мңјлЎң мһҗлҸҷ нҠёлҰ¬кұ°
  • GitHub Appмқҙ м ‘к·ј к°ҖлҠҘн•ң лҰ¬нҸ¬м§ҖнҶ лҰ¬лҘј Nixbotмқҙ discovery
  • мІ« import л•Ң build-with-buildbot нҶ н”Ҫ лҰ¬нҸ¬м§ҖнҶ лҰ¬ enable
  • мқҙнӣ„м—җлҠ” мӣ№ UIм—җм„ң adminмқҙ project enable/disable
  • PR мғқм„ұ/м—…лҚ°мқҙнҠёмҷҖ default branch push л•Ң .#checks нҸүк°Җ/л№Ңл“ң

к¶Ңн•ң¶

н•ӯлӘ© к°’ м„Өм • мң„м№ҳ
л№Ңл“ң лҢҖмғҒ SBEE-Lab мЎ°м§Ғ, mulatta мӮ¬мҡ©мһҗ modules/buildbot/master.nix: github.userAllowlist
мӣ№ кҙҖлҰ¬мһҗ github:mulatta modules/buildbot/master.nix: admins
мқёмҰқ GitHub OAuth services.nixbot.github.oauth*

кҙҖл Ё мӢңнҒ¬лҰҝ (modules/buildbot/secrets.yaml, sops м•”нҳёнҷ”):

мӢңнҒ¬лҰҝ мҡ©лҸ„
github-app-private-key GitHub App мқёмҰқ
github-oauth-secret мӣ№ UI лЎңк·ёмқё
github-webhook-secret мӣ№нӣ… HMAC кІҖмҰқ
niks3-auth-token м„ нғқ лҰ¬нҸ¬м§ҖнҶ лҰ¬ мҷёл¶Җ мәҗмӢң н‘ёмӢң

GitHub App м„Өм •¶

GitHub App м„Өм •мқҖ Nixbot нҳ•мӢқмңјлЎң мң м§Җн•ҙм•ј н•©лӢҲлӢӨ.

н•ӯлӘ© к°’
Webhook URL https://buildbot.sjanglab.org/webhooks/github
OAuth callback https://buildbot.sjanglab.org/auth/github/callback
Repository permissions Contents: Read & write, Checks: Read & write, Issues: Read & write, Metadata: Read-only, Pull requests: Read & write
Events Push, Pull request, Check run, Check suite

Contents м“°кё° к¶Ңн•ңмқҖ docs effectмқҳ gh-pages pushмҷҖ updaterмқҳ update branch pushм—җ н•„мҡ”н•©лӢҲлӢӨ. Pull requests м“°кё° к¶Ңн•ңмқҖ updater PR мғқм„ұм—җ, Issues м“°кё° к¶Ңн•ңмқҖ auto-merge label м¶”к°Җм—җ н•„мҡ”н•©лӢҲлӢӨ. к¶Ңн•ңмқ„ ліҖкІҪн•ҳл©ҙ к°Ғ installationм—җм„ң мғҲ к¶Ңн•ң мҠ№мқёмқҙ н•„мҡ”н•©лӢҲлӢӨ.

кҙҖлҰ¬мһҗ ліҖкІҪ¶

Nixbot кҙҖлҰ¬мһҗлҘј ліҖкІҪн•ҳл Өл©ҙ:

  1. modules/buildbot/master.nixм—җм„ң admins лӘ©лЎқ мҲҳм • (github:<login> нҳ•мӢқ)
  2. GitHub App м„Өм •м—җм„ң мЎ°м§Ғ/мӮ¬мҡ©мһҗ к¶Ңн•ң м—…лҚ°мқҙнҠё
  3. OAuth мӢңнҒ¬лҰҝ к°ұмӢ  н•„мҡ” мӢң sops modules/buildbot/secrets.yamlлЎң нҺём§‘
  4. inv deploy --hosts psi

л№Ңл“ң/effect мһ¬нҠёлҰ¬кұ°¶

мӢӨнҢЁн•ң л№Ңл“ңмҷҖ effectлҠ” Nixbot мӣ№ UIм—җм„ң мҲҳлҸҷмңјлЎң мһ¬нҠёлҰ¬кұ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ. https://buildbot.sjanglab.orgм—җ GitHub кі„м •мңјлЎң лЎңк·ёмқён•ң л’Ө, н•ҙлӢ№ л№Ңл“ң нҺҳмқҙм§Җм—җм„ң мһ¬мӢңмһ‘ лІ„нҠјмқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

Hercules-style effects¶

лҰ¬нҸ¬м§ҖнҶ лҰ¬мқҳ flake.herculesCIк°Җ Nixbot effectлҘј м •мқҳн•©лӢҲлӢӨ. NixbotмқҖ effect мӢӨн–ү мӢң GitHub App installation tokenмқ„ GitToken secretмңјлЎң м „лӢ¬н•ҳкі , effectлҠ” мқҙ tokenмңјлЎң git pushмҷҖ GitHub CLI мһ‘м—…мқ„ мҲҳн–үн•©лӢҲлӢӨ.

Effect Trigger лҸҷмһ‘
docs-pages main push .#docs кІ°кіјлҘј gh-pages лёҢлһңм№ҳлЎң force-pushн•©лӢҲлӢӨ. GitHub Pages sourceлҠ” terraform/github/repo.tfм—җм„ң gh-pages /лЎң кҙҖлҰ¬н•©лӢҲлӢӨ.
update-packages л§Өмқј 03:00 UTC .#updater -- --prлҘј мӢӨн–үн•ҙ updateable packageлі„ PRмқ„ мғқм„ұн•©лӢҲлӢӨ.

мҷёл¶Җ мәҗмӢң н‘ёмӢң¶

NixbotмқҖ mulatta/dots, mulatta/seqtable л№Ңл“ң м„ұкіө кІ°кіјл§Ң https://niks3.mulatta.ioлЎң pushн•©лӢҲлӢӨ. м „мІҙ л№Ңл“ңлҠ” psiмқҳ Harmonia cacheм—җм„ң кі„мҶҚ м ңкіөлҗ©лӢҲлӢӨ.

Package мһҗлҸҷ м—…лҚ°мқҙнҠё¶

Nixbot scheduled effectк°Җ л§Өмқј 03:00 UTCм—җ .#updater -- --prлҘј мӢӨн–үн•©лӢҲлӢӨ. UpdaterлҠ” packages/*/nix-update-args лҳҗлҠ” packages/*/update.pyлҘј л°ңкІ¬н•ҙ packageлі„ update branchмҷҖ PRмқ„ л§Ңл“ӯлӢҲлӢӨ. нҳ„мһ¬ slack-cliк°Җ nix-update/GitHub releases кё°л°ҳ м—…лҚ°мқҙнҠё лҢҖмғҒмңјлЎң л“ұлЎқлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

мғқм„ұлҗң PRм—җлҠ” auto-merge labelмқҙ л¶ҷкі , кё°мЎҙ auto-merge мӣҢнҒ¬н”ҢлЎңмҡ°к°Җ CI м„ұкіө л’Ө squash mergeн•©лӢҲлӢӨ.

Flake мһ…л Ҙ мһҗлҸҷ м—…лҚ°мқҙнҠё¶

Dependabotмқҙ л§Өмқј flake.lockмқ„ кІҖмӮ¬н•ҳм—¬ flake input мөңмӢ  м»Өл°Ӣ PRмқ„ мғқм„ұн•©лӢҲлӢӨ.

нқҗлҰ„: flake.lock ліҖкІҪ вҶ’ PR мғқм„ұ вҶ’ мһҗлҸҷ squash лі‘н•© вҶ’ mainм—җ л°ҳмҳҒ вҶ’ к°Ғ нҳёмҠӨнҠёк°Җ л§Өмқј 04:40 KSTм—җ system.autoUpgradeлЎң м Ғмҡ©. л§Өмӣ” л§Ҳм§Җл§ү нҶ мҡ”мқјм—җлҠ” auto-rebootк°Җ м Ғмҡ©лҗң м»Өл„җкіј л¶ҖнҢ… мӨ‘мқё м»Өл„җмқ„ л№„көҗн•ҳкі , ліҖкІҪлҗң кІҪмҡ° 24мӢңк°„ нӣ„ мһ¬л¶ҖнҢ…мқ„ мҳҲм•Ҫн•©лӢҲлӢӨ.

NixbotмқҖ flake м—…лҚ°мқҙнҠёмҷҖ л¬ҙкҙҖн•©лӢҲлӢӨ. NixbotмқҖ PR CI л№Ңл“ңл§Ң лӢҙлӢ№н•ҳкі , flake мһ…л Ҙ м—…лҚ°мқҙнҠё PR мғқм„ұмқҖ Dependabotмқҙ лӢҙлӢ№н•©лӢҲлӢӨ.
